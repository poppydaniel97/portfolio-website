Virtual House A Web Extension of Virtual Exhibitions

  • ROLE | UX & UI Design
  • DURATION | 1 Month
  • PLATFORMS | Web
  • TOOLS | Sketch, FlowMapp, InVision

ABOUT

Virtual House has been designed as a web extension of Virtual Exhibitions. A website that allows users to find and buy art that they love by upcoming artists.

STYLE

Much like Virtual Exhibitions, from the offset, I wanted to create a clear and sophisticated website. Consequently, I used a reduced colour palette and modern typography.

SEARCH

The user will find artwork for sale by using the search feature. Users will be able to refine their search by using the filter feature.
